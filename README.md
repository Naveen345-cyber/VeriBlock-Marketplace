# VeriBlock Marketplace - Mall Data Exchange

Built for CGC Mohali Hackathon 2026.

## Prerequisites

* Ganache (Port 7545)
* IPFS Desktop / Kubo Daemon
* Java 17+ & Maven

## Setup Instructions

1. **Blockchain**: Open Ganache on port 7545.
2. **Backend**:
    * Navigate to `/backend`
    * Run `mvn spring-boot:run`
3. **Frontend**:
    * Navigate to `/frontend`
    * Run `npm install`
    * Run `npm start`

## Tech Stack

* **Smart Contracts**: Solidity (Stablecoin: VeriINR)
* **Backend**: Spring Boot, Web3j
* **Storage**: IPFS
* **Frontend**: React.js
